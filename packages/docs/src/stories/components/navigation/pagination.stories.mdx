# Pagination

Pagination is a component to navigate each page. usually, use when we have enormous data and want to separate the data before we are showing

## Example Usage

```tsx
import { Pagination } from '@01group-ui/core';
```

## Example Pagination

```tsx
export const ExamplePagination = () => {
  const [currentPage, setCurrentPage] = useState(1);
  return (
    <Pagination
      totalCount={1000}
      pageSize={10}
      siblingCount={2}
      currentPage={currentPage}
      onChangePage={(page: number) => setCurrentPage(page)}
      showSizeChange={true}
      listCustomSizePage={[10, 20, 50, 100]}
    />
  );
};
```

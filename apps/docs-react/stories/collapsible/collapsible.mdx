# Collapsible

An interactive component that expands/collapses a panel.

## Example

```tsx
import { Collapsible, CollapsibleContent, CollapsibleTrigger } from '#/components/base-ui'

<Collapsible>
  <CollapsibleTrigger>Toggle</CollapsibleTrigger>
  <CollapsibleContent>
    <div>Collapsible content</div>
  </CollapsibleContent>
</Collapsible>

// With custom trigger
<Collapsible>
  <CollapsibleTrigger asChild>
    <Button>Toggle</Button>
  </CollapsibleTrigger>
  <CollapsibleContent>
    <div>Content</div>
  </CollapsibleContent>
</Collapsible>
```
